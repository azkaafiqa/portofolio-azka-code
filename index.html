Tentang Saya

Nama saya Azka, dan di sekolah, saya dikenal sebagai seseorang yang memiliki ketertarikan besar pada dunia teknologi. Namun, ada satu pengalaman yang benar-benar menguji mental dan logika saya, yaitu saat saya harus menyelesaikan sebuah proyek coding yang cukup rumit.

Pengalaman Saya

Semuanya bermula ketika saya ditantang untuk membuat website sederhana sebagai tugas akhir mata pelajaran Informatika. Awalnya, saya merasa sangat bersemangat. Membayangkan program yang saya buat bisa berjalan dengan lancar rasanya sangat membanggakan. Namun, realitanya ternyata tidak semudah menekan tombol run.

Tantangan terbesar muncul saat saya menghadapi ribuan baris kode yang tiba-tiba mengalami error. Saya ingat betul, ada satu malam di mana saya menghabiskan waktu berjam-jam hanya untuk mencari satu kesalahan kecil, seperti titik koma (;) yang terlewat atau kurung kurawal ({}) yang tidak tertutup. Rasanya sangat frustrasi. Rasanya ingin menyerah saja dan mengumpulkan tugas seadanya.

Namun, saya teringat bahwa nama saya adalah Azka, dan seorang programmer sejati tidak akan membiarkan bug menang. Saya mulai belajar untuk lebih teliti. Saya mencari referensi di forum-forum komunitas, menonton tutorial, dan bertanya kepada guru. Saya belajar bahwa error bukanlah kegagalan, melainkan cara program memberi tahu saya bahwa ada sesuatu yang perlu diperbaiki.

Momen yang paling membahagiakan adalah ketika kalimat "Process finished with exit code 0" atau tampilan program yang saya inginkan akhirnya muncul di layar. Rasa lelah berjam-jam itu hilang seketika, digantikan oleh rasa bangga yang luar biasa.

Dari proyek sulit ini, saya, Azka, belajar banyak hal melebihi sekadar bahasa pemrograman. Saya belajar tentang logika berpikir kritis, ketelitian, dan yang terpenting adalah kerendahan hati untuk terus belajar. Coding mengajarkan saya bahwa masalah besar bisa diselesaikan jika kita memecahnya menjadi bagian-bagian kecil yang logis.
